Vercel 部署指南

本网页是一个纯静态 HTML 文件，可以直接部署到 Vercel 实现公网访问。

步骤 1: 准备代码

将本文件保存为 index.html。

步骤 2: 上传到 Vercel

  1. 登录 Vercel。
  2. 点击 "Add New..." -> "Project"。
  3. 如果你有 GitHub 仓库，可以直接导入；或者直接将包含 index.html 的文件夹拖拽到 Vercel 的 Dashboard 中（如果使用 CLI）。
  4. 最简单的方法：安装 Vercel CLI (npm i -g vercel)，在文件所在目录运行 vercel，按回车确认即可。

关于数据安全

注意： 本应用是纯前端应用，API Key 存储在浏览器的 LocalStorage 中。这意味着：
  • 不要在公共电脑上使用。
  • 如果你将代码开源，不要在代码中硬编码 API Key。
  • 每次在新浏览器打开，都需要重新在"配置中心"输入 Notion API Key 和数据库 ID。
